# Conditions (else if)

### if - else sytax

```
if () {
    alert()
} else {
    alert()
}
```

### else if syntax

```
if ([conditions]) {
    alert()
} else if (conditions) {
    alert()
} else {
    alert()
}
```

### Example: 

```
if (2 === 2) {
    alert("hi")
} else if (3 === 3) {
    alert("bye")
} else {
    alert("welcome")
}
```
