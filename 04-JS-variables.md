# Variables

### How to write:
`var [variable-name] = .....`

### Example: 
`var UserAccount = 13`

### How to use:
`....(UserAccount)`

### Example: 
`alert(UserAccount)`

### Important note:
  **In inspect into console tab we can see our bugs or errors.**
