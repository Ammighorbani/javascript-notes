# isNaN 

### Note: 
**This method will check our operation is it NaN or not if it's NaN we will receive "true" and it's not we will reveive "false".**

### Example: 

**Inpute:**
```
alert(isNaN("hi" - 2))
```

**Output:**
```
true
```
