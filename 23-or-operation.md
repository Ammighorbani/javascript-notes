# "or" Operation 
### "or" model:  "||"

### | : It's pipe line

### Note: 
**"or" look for two or more condition and searching for "true" condition if all of our conditions be "false" it won't enter into our scope.**

### Example:

```
if (2+2 === 4 || 3+3 === 6) {
    alert("welcome")
} else if (2+2 === 6 || 3+3 === 2) {
    alert("bye")
}
```

### Note:

```
"true", "true" ==> "true"

"true", "false" ==> "true"

"false", "false" ==> "false"
```
