# Ternary Condition 

### we can create if in only one line

### Ternary Condition Syntax:
```
var resault = condition ? value1 : value2;
```

### Example: 
```
var userAge = (prompt("Enter your age:") > 18) ? true : false;
alert(userAge)
```
