# Conditions (if-else)

### "if" : 
**We use it to create conditions in our codes.**

### if syntax:
```
if ([condition]) {
	[codes]
}
```

### Example:
```
if (2===2) {
    alert("true")
}
```

### "else" : 
**We use else to create second condition and when "if" is incorrcet "else" will happen.**

### else sytax:
```
else {
    [codes]
}
```

### Example:
```
if (3===2) {
    alert("true")
} else {
    alert("false")
}
```
