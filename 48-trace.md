# Trace

### We can create something like debugger like this.

### Syntax:

```
 var1	| var2	| .....
---------------------------
 .....	| .....	| .....
 .....	| .....	| .....
 .....	| .....	| .....
 .....	| .....	| .....
```

### Example:

```
    i	|  name	| log(name[i])
------------------------------
      |  Amir	 | 
    0	|	       | log(name[0]) ==> A 
    1	|	       | log(name[0]) ==> m
    2	|	       | log(name[0]) ==> i
    3 |        | log(name[0]) ==> r
```

### Note: 
**It will help us to debug our codes.**
