# "for" Loop Part-2

### Example:

**Inpute:**

```
for (var i = 3; i>0; i--){
	console.log(i)
}
```

**Output:**

```
3
2
1
```

### Example: 
**Get our "String" index with "for" loop.**

**Inpute:**

```
var userName = "Amir Mahdi"

for (var i = 0; i < userName.lenght; i++) {
	console.log(userName[i])
}
```

**Output:**

```
A
m
i
r

M
a
h
d
i
```

### Note: 
**Our condition must be "true" to our loop start.**
