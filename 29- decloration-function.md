# Declaration Function:

### Syntax:
```
function [func-name] () {
    [code]
}
```

How to call the function: `[function-name]()`

### Example:
```
function sums () {
    var num1 = 4
    var num2 = 3
    alert(num1 + num2)
}

sums()
```

### Note: 
**We call "declaration function" that function we wrote it with "function" key word.**
