# Expression Function

### Note: 
**It's completely look like declaration functions.**

### Syntax:
```
var [var-name] = function () {
            code
        }
```

How to call: `[var-name]()`

### Example:
```
var showVariable = function () {
            var num1 = 3
            var num2 = 4
            alert(num1+num2)
        }

showVariable()
```
