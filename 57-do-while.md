# "do-while" Loop

### Sntax:
```
do {
	[codes]
} while ([condition])
```

### Example:

**Inpute:**
```
var i = 0

do {
	alert(1)
} while (i > 1)
```

**Output:**
```
1
```

### Note: 
**In "do-while" loop one time our condition will start and then our "condition" will check anyway one time our code will start.**
