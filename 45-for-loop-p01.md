# "for" Loop Part-1

### syntax:

```
for (begin; condition; step) {
	[codes]
}
```

### Example:

**Inpute:**

```
for (var i = 0; i < 5; i++) {
	console.log(i, "yes")
}
```

**Output:**

```
0 "yes"
1 "yes"
2 "yes"
3 "yes"
4 "yes"
```
